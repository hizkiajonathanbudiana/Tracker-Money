## ğŸ› Troubleshooting

**Install button doesn't show?**
- PWAs only work on HTTPS or localhost
- Check if already installed
- Try incognito/private browsing mode

**App not working offline?**
- Make sure you built for production: `npm run build`
- Service worker disabled in dev mode
- Try clearing cache and reinstalling

**Icons not showing?**
- Run `node generate-icons.js`
- Check icons exist in `public/` folder
- Clear browser cache
